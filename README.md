# I2Solutions Aplicación móvil

## Development

1. Instalar en global ionic cli

```sh
npm i -g ionic cordova
```

2. Instalar dependencias

```sh
npm install
```

3. Correr la aplicación

```sh
 ionic serve --lab -c
```

ionic cordova run android
## Production

 Build android .apk

```sh
ionic cordova build android
```
